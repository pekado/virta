# PhoneInput

`PhoneInputField` is a specialized TextInput that auto-formats user inputs as phone numbers.
This component should be used within a Formik form, and currently requires inputs to take
the form 999-999-9999, but could be easily extended to use other formats, or even a
prop-provided format.

# Table

A couple notes about using this `Table` component:

- `TableRow` is able to display `children` and/or an array of `data` â€” children will be rendered first,
  and then it maps each item in the array as a `TableCell`.

- `TableCell` is pretty flexible in what it accepts as a `value` prop so you can pass it a string
  or another component.
